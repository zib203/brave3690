# Работа с удаленными репозиториями

Копирование всех файлов и коммитов удаленного репозитория 
```sh
git clone <ссылка>
```
Выгрузка локальной репозитории на сервер
```sh
git push
```
Загрузка репозитории с сервера
```sh
git pull
```
